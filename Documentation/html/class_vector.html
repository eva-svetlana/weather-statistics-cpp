Detailed Description

template<typename T>
class Vector< T >

An expandable array that handles memory and lets you store any type.

Author
Svetlana Alkhasova
Date
18/07/25
Version
2.0

This is a basic custom vector class — it can grow if you add more items, and lets you safely get/set values with []. Good for assignments where you can’t use the STL vector by itself.

Template Parameters
TThe type of things you want to store (like int, float, WeatherEntry, ...)
